# Setup Guide

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/GikOnFlik/openapi-tw.git
   ```
2. Navigate into the project directory:
   ```bash
   cd openapi-tw
   ```
3. Install required dependencies:
   ```bash
   npm install
   ```

## Configuration Instructions
1. Create a `.env` file in the root directory of the project.
2. Add your configuration settings into the `.env` file:
   ```
   API_KEY=your_api_key_here
   DB_HOST=your_database_host
   DB_USER=your_database_user
   DB_PASS=your_database_password
   ```
3. Save the file and ensure it is included in your `.gitignore` to avoid committing sensitive information.
